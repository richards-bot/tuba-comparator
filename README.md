# Tuba Comparator

A clean comparison site for tubas: qualities, price, and where to buy.

## Features
- Filter by key and player level
- Sort by price
- Click a tuba card for full specs and shop links
- Mobile-friendly layout

## Run
```bash
npm install
npm run dev
```

## Deploy
Push to `main` and GitHub Pages deploys automatically via `.github/workflows/deploy-pages.yml`.
